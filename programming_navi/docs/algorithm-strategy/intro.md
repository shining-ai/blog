# アルゴリズム戦略

アルゴリズム戦略とは、問題を効率的に解くための**設計パラダイム**です。
個別のアルゴリズムとは異なり、多様な問題に横断的に適用できる汎用的な思考の枠組みです。

## 主要戦略の比較

| 戦略 | 核心原則 | 適用条件 | 計算量の傾向 |
| --- | --- | --- | --- |
| 分割統治法 | 問題を分割し再帰的に解く | 部分問題が独立 | O(n log n) |
| 貪欲法 | 局所最適選択を繰り返す | 貪欲選択性 + 最適部分構造 | O(n log n) |
| 動的計画法 | 部分問題をメモ化して解く | 重複部分問題 + 最適部分構造 | O(n²) 〜 O(nW) |
| バックトラッキング | 候補を構築し制約違反で枝刈り | 制約充足問題 | 指数時間（枝刈りで改善） |
| 乱択アルゴリズム | 乱数で期待性能を改善 | 最悪ケース回避が必要 | 期待多項式時間 |
| 近似アルゴリズム | 保証付き近似解を多項式時間で | NP 困難問題 | 多項式時間 |
| オンラインアルゴリズム | 逐次入力を即座に処理 | 将来の入力が未知 | 問題依存 |

## 戦略の選び方

```
問題が最適解を持ち、部分問題が独立している
  └─ 分割統治法

問題が最適解を持ち、部分問題が重複する
  ├─ 局所最適選択が全体最適になる → 貪欲法
  └─ ならない              → 動的計画法

解の列挙・制約充足が必要
  └─ バックトラッキング

決定的アルゴリズムの最悪ケースが問題
  └─ 乱択アルゴリズム

NP 困難問題で最適解は不要（近い解でよい）
  └─ 近似アルゴリズム

将来の入力が未知でリアルタイム処理が必要
  └─ オンラインアルゴリズム
```
